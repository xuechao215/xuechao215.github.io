前端开发之准备

2017-02-06

服务器与浏览器之间的关系

服务器与浏览器的基本概念

  • 什么是浏览器
    • 可以显示网页服务器或者文件系统的HTML文件内容,并让用户与这些文件交互的一种软件
    • 输入网址发送请求报文到服务器
  • 什么是服务器
    • 提供计算服务的设备
      • web服务器
      • 文件服务器
      • 邮件服务器
      • FTP服务器
    • 响应请求

浏览器的内核

  • 四种主要内核
    • Trident: ie ,猎豹安全浏览器,360极速浏览器,百度浏览器
    • Gecko: Firefox
    • Webkit: Safari
    • Blink: Chrome, Opera

HTTP协议

  • 浏览器与服务器交互的约定:HTTP协议
  • HTTP协议:请求报文 响应报文 URL
  • URL:
    • 协议名
    • 服务器IP地址
    • 端口号
    • 请求文件的名称
  • 域名:
    • DNS服务器:域名解析系统
    • DNS服务器将域名解析成IP地址
  • 默认端口:80

编辑器Sublime的使用

  • ctrl + k + b 隐藏显示侧边栏
  • ctrl + n 新建文件
  • ctrl + shift + d 快速复制一整行
  • ctrl + shift + k 快速删除一整行
  • ctrl + shift + ↑ 向上移动一整行
  • ctrl + shift + ↓ 向下移动一整行

路径问题

  • 绝对路径:带有盘符的路径
  • 相对路径:图片相对页面的路径
