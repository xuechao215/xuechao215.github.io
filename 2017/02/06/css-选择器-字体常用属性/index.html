css--选择器,字体常用属性

2017-02-06

CSS(Cascading Style Sheets):层叠样式表

三层分离标准

  • 三层
    • html:页面结构–从语义角度描述页面结构
    • css:页面样式–从样式上修饰结构
    • JavaScript:行为–页面交互和动态效果
  • 语义的好处
  1. 让页面更加友好,有利于SEO
  2. 页面的结构更加清晰,对开发人员友好
  3. 对用户体验比较好

CSS初体验

  • 注意:所有css相关的设置都放在style标签中,style标签是一个双标签,放在head标签中,title标签下面
  • 结构

CSS相关属性

  • color:文字颜色
  • font-size:文字大小
  • background-color:背景颜色
  • width:宽
  • height:高

与字体相关的属性

  • font-size:字体大小
  • font-weight:字体粗细
    • 可以设置100-900之间的整百数字
    • 可以设置关键字: blod:加粗 normal:正常
  • font-family:字体类型
    • “Microsoft yahei”:微软雅黑
    • “simsun”:宋体
  • font-style:字体样式
    • normal:正常
    • italic:斜体

字体连写

  • 格式

*注意:font-style font-weight 可以不写

颜色表示法

  • color 文字颜色
  • 取值
  1. 具体的英文颜色单词
    • pink white red green blue
  2. 16进制
    • #000000 ->#000
    • #003300 ->#030
    • FFFFFF ->#FFF
  3. RGB表示法
    • rgb(red,green,blue)
    • rgb(0,0,0) ~ rgb(255,255,255)
  4. RGBA表示法
    • rgba(red,green,blue,alpha)

选择器

标签选择器

  • 作用:将这一类标签加上相同的css样式

类选择器

  • 使用步骤:先给要设置样式的标签添加类名,再使用类选择器设置样式
  • 作用:给设置了这个类名的那些标签都添加上样式

id选择器

  • 使用步骤:先给要设置样式的标签设置id,再使用id选择器设置样式
  • 给设置了这个id的标签添加样式

类选择器与id选择器的区别

  • 类名:类比人的姓名
    • 可以设置给多个类名
    • 一个标签可以有多个类名
  • id名:类比人的身份证号码
    • 只能给一个标签设置这个id
    • 一个标签只能有一个id
      注意:id不是给css使用的,是留给JavaScript

选择器的命名规则

  • 命名的取值范围:0~9,a~z,A~Z,_,-
  • 不能以数字开头

通配符

  • 作用:会将页面上所有的标签都添加上设置的样式

交集选择器

  • 注意:如果交集选择器中有标签选择器，则标签选择器必须写在前面

子代选择器

  • 作用:找到一个标签的直接子元素来设置样式

css代码的存放位置

  • 行内样式

  • 嵌套样式
    *css代码以style标签的方式写在head标签内

  • 外联样式
    • 新建一个css文件，在其中写好样式代码，不需style标签
    • 在head标签内引用这个css文件
