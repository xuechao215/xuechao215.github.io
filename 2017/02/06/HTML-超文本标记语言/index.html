HTML--超文本标记语言

2017-02-06

HTML:超级文本标记语言(HyperText Markup Language),在浏览器上运行的一种标记语言.

标签简介

基本模式

DOCTYPE

  • DOCTYPE:文档类型
    • html
    • xhtml
  • DTD:文档类型定义
  • HTML&XHTML都有三种规范
  • 使用HTML5规范

SEO相关

  • SEO:搜索引擎优化
    • 影响网站排名
  • SEO方式
    • 付费
    • 静态页面
    • 发外链推广
    • 页面友好–规范
      *合理使用标签的语义化

Head标签

  • 用于存放:title(必选),meta,style,script,link

meta标签的用法

  • title标签的用法及作用
    • 作用:title在SEO中占有很大的权重
  • meta标签中的内容不会在页面显示

meta设置字符编码

  • 字库
    • GB2312:国标2312
    • GBK:国标扩
    • utf-8:收录全世界所有语言中的文字
  • UTF-8存储一个汉字占3个字节
  • GBK存储一个汉字占2个字节

标签的分类

  1. 双标签:有开始有结束,开始与结束之间有内容
  1. 单标签:只有一个标签,自己闭合自己

列表

无序列表

  • ul标签中嵌套li标签
    • ul:unordered list
    • li:list item
  • li标签中可以嵌套其他标签

有序列表

  • ol:ordered list

自定义列表

  • 作用:显示一段格式自定义的数据

表格

  1. 标签:
  • table:表格
  • tr:行
  • td:单元格
  • th:表头的内容
  • caption:表格的标题
  • tbody:表格的主体
  • thead:表头的表示
  • tfoot:表尾的表示
  1. 属性:
  • border:边框
  • width:表格的宽
  • height:表格的高
  • cellspacing:单元格之间的距离
  • cellpadding:单元格中内容与边距的距离

表单

  • 输入框

  • 密码框

  • 隐藏起来的输入框

  • 单选框

  • 复选框

  • 按钮

  • 图片按钮:可以点击的图片

  • 下拉框

  • 文本域

其他标签
